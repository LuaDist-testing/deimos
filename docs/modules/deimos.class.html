Module deimos.class

Deimos 'class' type.

Implements an object with fixed key-value pairs, much like a class in other languages.

x.validate (self, data)
Tests that 'data' is an object of the same type described by 'schema'.

Parameters:

  • self: Schema object, the type being checked for
  • data: Data object being tested.

Returns:

    Nothing, unless the object was a valid class, in which case it is filled(). There are no negative cases: errors are thrown if the data is invalid.

Parameters:

  • self: Type of object being filled
  • data: Data to be filled in/validated

Returns:

    A complete class of the given type in 'self', with it's contents set to the contents of 'data'. In the case of a validation error, nil + error message is returned instead.
